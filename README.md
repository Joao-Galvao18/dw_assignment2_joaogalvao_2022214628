## AnimeVerse

AnimeVerse is a responsive anime showcase website combining visual design and dynamic content using the Jikan API.

The website brings together the latest and greatest anime through a
cinematic interface. Powered by the Jikan API, it
dynamically fetches real-time data from MyAnimeList so you can discover
the shows that are trending and explore by genres.

## Live Demo
[View on GitHub Pages](https://joao-galvao18.github.io/dw_assignment2_joaogalvao_2022214628/)

## Features
- Cinematic landing page with looping videos
- Scroll-reactive and theme-adaptive navbar
- Dark/light mode with custom icons
- Real-time anime data from the Jikan API
- Smooth animations and responsive design

## Built With
- HTML  
- CSS (Flexbox, Grid, Transitions)  
- JavaScript (Fetch API, Scroll Events)  
- [Jikan API](https://docs.api.jikan.moe/)
